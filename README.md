TAGGER.JS
=========

Tagger.js is a jQuery plugin for tagging models

DEPENDENCIES
============

* jQuery (tested with 1.4.2)
* jQueryUI (tested with 1.8.2)
* jQueryUI-autocomplete

USAGE
=====

html:

    <ul id="user_tags"></ul>

javascript:

    $(function() { $("user_tags").tagger(); });


PARAMETERS

* source: an array with the pre-existent tags
